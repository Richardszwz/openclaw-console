## 快速开始

### 启动控制台

```bash
# 启动
openclaw console start

# 停止
openclaw console stop

# 重启
openclaw console restart

# 查看状态
openclaw console status
```

### 访问控制台

默认地址：**http://localhost:4310**

如果自定义了端口，请使用你配置的端口访问。

### 认证

默认情况下，OpenClaw Console 复用 OpenClaw Gateway 的认证机制：

- 如果 OpenClaw Gateway 需要 Token 认证，控制台会自动传递
- 默认只允许 localhost 访问，外网访问需要配置 `BIND=0.0.0.0`

## 网关监控中心 (Gateway Hub)

### 功能概述

Gateway Hub 是 OpenClaw 核心网关的监控和管理中心，你可以在这里：
- 查看网关实时运行状态
- 监控性能指标
- 查看实时日志
- 一键启停网关

### 界面说明

#### 1. 仪表盘

**截图说明**：
- 顶部是四个状态卡片：运行状态、版本、运行时长、活跃会话
- 中间是性能指标图表：QPS（每秒查询率）、延迟（p50/p95/p99）、内存占用
- 右侧是渠道连接状态卡片：显示微信、飞书等各渠道是否在线
- 底部是最近告警列表

**操作说明**：
- 鼠标悬停在图表上可以查看详细数值
- 点击渠道卡片可以查看该渠道的详细配置

#### 2. 日志中心

**截图说明**：
- 左侧是日志级别筛选器（Debug/Info/Warn/Error）
- 顶部是搜索框和时间范围选择器
- 中间是实时日志流，最新日志在底部自动滚动
- 右侧是日志统计柱状图

**操作说明**：
1. **搜索日志**：在搜索框输入关键字，按回车搜索
2. **筛选级别**：点击左侧日志级别复选框筛选
3. **暂停滚动**：点击「暂停」按钮停止自动滚动
4. **导出日志**：点击「导出」按钮导出当前筛选结果为 JSON 文件
5. **清空**：点击「清空」按钮清空当前视图（不删除实际日志）

#### 3. 操作面板

**截图说明**：
- 三个大按钮：启动、停止、重启
- 一个配置热更新按钮
- 紧急停止按钮（红色）

**操作说明**：
- **启动**: 启动 OpenClaw Gateway（如果已停止）
- **停止**: 停止正在运行的 Gateway
- **重启**: 先停止再启动
- **热更新**: 重新加载配置文件，不需要重启
- **紧急停止**: 立即停止 Gateway 和所有活跃会话（危险操作，需要二次确认）

### 常用操作流程图

```
查看状态 → 发现异常 → 查看日志定位问题 → 
  ├─ 配置错误 → 修改配置 → 热更新 → 恢复正常
  └─ 进程卡住 → 重启网关 → 恢复正常
```

## 智能体舰队 (Agent Fleet)

### 功能概述

Agent Fleet 管理所有配置的 AI Agent，你可以：
- 查看所有 Agent 的运行状态
- 启停单个 Agent
- 修改 Agent 配置
- 查看 Agent 专属日志
- 监控每个 Agent 的负载

### 界面说明

#### 1. Agent 列表

**视图切换**：
- **卡片视图**：可视化展示，适合概览
- **表格视图**：详细数据，适合批量查看

**列/卡片信息**：
- Agent 名称和 ID
- 当前状态（运行中/已停止/错误）
- 使用的模型
- 已启用技能数量
- 内存占用
- 最近活动时间

**操作说明**：
- **状态筛选**：使用顶部筛选栏按状态筛选
- **搜索**：在搜索框输入 Agent 名称搜索
- **点击卡片**：进入 Agent 详情页
- **快速操作按钮**: 重启/查看日志

#### 2. Agent 详情页

**截图说明**：
- 顶部：基本信息卡片（名称、ID、模型、工作目录）
- 左侧：已启用技能列表，显示每个技能的版本和描述
- 中间：性能统计图表（会话数趋势、Token 使用量）
- 右侧：最近 10 个会话列表
- 底部：配置编辑器

**操作说明**：
1. **编辑配置**：点击「编辑配置」按钮，修改技能列表、模型等配置
2. **保存配置**: 点击保存后自动生效（需要重启 Agent）
3. **重启 Agent**: 点击「重启」按钮重新加载配置
4. **查看历史**: 点击历史会话可以跳转到会话详情

#### 3. 技能权限配置

**功能说明**：
- 每个 Agent 可以启用/禁用特定技能
- 可以配置技能的访问权限
- 支持技能版本锁定

**操作说明**：
- 在 Agent 详情页的技能列表中，勾选要启用的技能
- 点击技能名称可以查看技能详情
- 在下拉框选择技能版本（锁定版本时固定）

### 使用场景

| 场景 | 操作步骤 |
|------|---------|
| 新增 Agent | 配置中心 → 添加 Agent → 在 Agent Fleet 查看状态 |
| Agent 无响应 | Agent 详情 → 查看日志 → 发现错误 → 修复配置 → 重启 |
| 更换模型 | Agent 详情 → 编辑 → 选择新模型 → 保存 → 重启 |

## 会话管理中心 (Session Manager)

### 功能概述

Session Manager 管理所有 AI 会话，你可以：
- 查看实时会话列表
- 筛选和搜索会话
- 查看完整对话历史
- 批量终止/删除会话
- 导出会话数据

### 界面说明

#### 1. 会话列表

**截图说明**：
- 顶部：筛选栏（Agent 筛选、状态筛选、渠道筛选）
- 搜索框：按会话 ID 或关键字搜索
- 列表：每一行显示会话 ID、Agent、状态、开始时间、耗时、Token 数
- 底部：分页控件和批量操作按钮

**操作说明**：
- **筛选**: 选择筛选条件后列表自动更新
- **搜索**: 输入关键字后按回车搜索
- **多选**: 勾选左侧复选框选中多个会话
- **批量操作**: 选中后点击「批量终止」或「批量删除」
- **刷新**: 点击刷新按钮手动刷新（支持自动刷新）

#### 2. 会话详情

**截图说明**：
- 顶部：会话基本信息卡片（ID、Agent、状态、耗时）
- Token 统计：输入 tokens、输出 tokens、总 tokens
- 中间：对话历史，用户消息在右侧，助手消息在左侧
- 右侧：元数据信息（模型、渠道、开始时间、更新时间）
- 如果有子会话（Subagent），显示关联会话导航树

**操作说明**：
- **终止会话**: 如果会话正在运行，点击「终止」按钮
- **删除会话**: 点击「删除」按钮删除这条记录（需要确认）
- **导出会话**: 点击「导出」导出为 JSON 文件
- **查看子会话**: 点击关联会话跳转到子会话详情

#### 3. 统计报表

**截图说明**：
- 顶部：时间范围选择器（今日/本周/本月/自定义）
- 左侧：数字卡片（总会话数、完成数、失败数、平均耗时）
- 右侧：折线图显示每日会话创建趋势

**操作说明**：
- 选择时间范围后图表自动更新
- 鼠标悬停显示详细数据

### 批量操作指南

1. **批量终止卡住的会话**：
   - 筛选状态为「运行中」超过 1 小时的会话
   - 全选 → 点击「批量终止」

2. **清理历史会话**：
   - 筛选状态为「已完成」超过 30 天的会话
   - 全选 → 点击「批量删除」（注意：删除不可恢复）

## 工作流工作室 (Workflow Studio)

### 功能概述

Workflow Studio 让你可视化设计和监控 AI 工作流，支持：
- 六 Agent/十 Agent 标准协作模板
- 拖拽式工作流设计
- 实时任务流监控
- 历史执行回放

### 界面说明

#### 1. 工作流列表

**截图说明**：
- 左侧：模板分类（标准模板/我的工作流）
- 右侧：工作流卡片，显示名称、状态、版本、最近执行时间

**操作说明**：
- **使用模板**: 点击模板卡片 → 「使用此模板」→ 保存为你的工作流
- **新建**: 点击「新建工作流」按钮创建空白工作流
- **编辑**: 点击工作流卡片进入编辑模式
- **复制**: 点击更多操作 → 复制创建副本

#### 2. 流程设计器

**截图说明**：
- 左侧：节点工具栏（Agent 节点/审批节点/条件节点/结束节点）
- 中间：拖拽画布，网格背景
- 右侧：属性面板，编辑当前选中节点的属性

**节点类型**：

| 节点类型 | 说明 |
|----------|------|
| Agent 节点 | 调用指定 Agent 处理 |
| 审批节点 | 需要人工审批通过才能继续 |
| 条件节点 | 根据条件分支走不同路径 |
| 结束节点 | 工作流结束 |

**操作说明**：
1. **添加节点**: 从左侧拖拽节点到画布
2. **连接节点**: 从一个节点的输出拖拽到另一个节点的输入创建连线
3. **删除节点/连线**: 选中后按 Delete 键
4. **编辑属性**: 选中节点后在右侧属性面板编辑
5. **保存**: 点击「保存」按钮保存工作流

#### 3. 流程监控

**截图说明**：
- 画布上：当前运行的任务用绿色高亮显示正在处理的节点
- 右侧：执行日志，显示每个节点的开始/结束时间和输出
- 底部：历史执行列表

**操作说明**：
- **启动工作流**: 点击「启动」按钮开始执行
- **暂停/继续**: 执行中可以暂停或继续
- **终止**: 点击「终止」停止执行
- **回放历史**: 点击历史记录 → 「回放」查看执行过程动画

### 标准模板说明

**六 Agent 协作流程**：
```
需求分析 → 架构设计 → 代码实现 → 代码评审 → 测试 → 文档
```

**十 Agent 协作流程**：
```
产品需求 → 用户研究 → UX 设计 → 架构设计 → 前端开发 → 
后端开发 → 代码评审 → 集成测试 → 文档编写 → 发布上线
```

## 任务调度中心 (Task Scheduler)

### 功能概述

Task Scheduler 管理定时任务，支持：
- 可视化编辑 Cron 表达式
- 任务执行历史查看
- 失败自动重试
- 失败告警通知

### 界面说明

#### 1. 任务列表

**截图说明**：
- 每一行显示：任务名称、Cron 表达式、下次执行时间、上次执行状态、操作按钮
- 状态用颜色标识：绿色正常，黄色错过执行，红色失败

**操作说明**：
- **启用/禁用**: 点击开关切换
- **立即执行**: 点击「立即执行」按钮手动触发
- **编辑**: 点击「编辑」进入编辑页面
- **查看日志**: 点击「日志」查看执行历史

#### 2. 任务编辑器

**截图说明**：
- 任务名称输入框
- 自然语言输入框（支持 "每天凌晨 3 点" 自动转换为 Cron）
- 可视化 Cron 编辑器（下拉选择分/时/日/月/周）
- 命令输入框（要执行的 openclaw 命令）
- 重试配置（失败重试次数、重试间隔）
- 告警配置（失败时是否告警、通知方式）

**操作说明**：
1. **快速设置**: 在自然语言框输入描述，点击「转换」自动生成 Cron
2. **精细调整**: 使用可视化编辑器调整 Cron
3. **预览**: 下方显示最近 5 次执行时间预览
4. **保存**: 点击保存生效

#### 3. 执行历史

**截图说明**：
- 列表显示最近 50 次执行记录
- 每一行显示执行时间、状态、耗时
- 点击展开可以查看完整执行输出日志

**常用场景**：

| 任务示例 | Cron 表达式 | 说明 |
|----------|-------------|------|
| 每日备份 | `0 3 * * *` | 每天凌晨 3 点执行备份 |
| 每周系统检查 | `0 2 * * 1` | 每周一凌晨 2 点检查 |
| 每月用量统计 | `0 1 1 * *` | 每月 1 号凌晨 1 点统计 |

## 模型用量监控 (Model Usage)

### 功能概述

Model Usage Dashboard 监控各大 LLM 平台的模型用量，支持：
- 多平台用量统计
- 实时显示今日/本月用量
- 用量趋势图表
- 总量配额预警
- 成本估算

### 支持平台

- Volcengine (Doubao)
- MiniMax
- OpenAI
- Anthropic
- Google Gemini
- 自定义平台

### 界面说明

#### 1. 模型配置

**截图说明**：
- 列表显示所有已配置的模型
- 每一行显示：模型名称、提供商、月度配额、已用百分比、启用状态
- 「添加模型」按钮

**操作说明**：
- **添加**: 点击「添加模型」→ 选择提供商 → 输入 API Key → 配置配额和价格 → 保存
- **编辑**: 点击列表右侧编辑按钮
- **同步**: 点击「从 openclaw.json 同步」自动导入配置

#### 2. 用量统计

**截图说明**：
- 顶部卡片：今日用量（输入/输出 tokens）、本月用量、估算成本
- 每个模型卡片：显示已用量、配额、百分比、今日成本
- 红色边框：超过预警阈值

**操作说明**：
- **刷新用量**: 点击「刷新」按钮从 API 拉取最新用量
- **设置预警**: 在模型配置中设置预警阈值（如 80%）

#### 3. 趋势图表

**截图说明**：
- 折线图：近 30 天每日用量趋势
- 饼图：各模型用量占比
- 柱状图：每日成本对比

**操作说明**：
- 切换时间范围（7天/30天/90天）
- 鼠标悬停查看具体数值

#### 4. 告警记录

**截图说明**：
- 列表显示所有触发的告警
- 包含：模型名称、触发时间、当前用量百分比、处理状态
- 支持标记为已确认

### 成本估算公式

```
估算成本 = (总 tokens / 1,000,000) × 每百万 tokens 价格
```

价格在模型配置中设置，可以根据实际价格调整。

## 系统配置中心 (Settings)

### 功能概述

Settings Center 统一管理系统配置，支持：
- OpenClaw 配置编辑
- 模型渠道配置
- 权限管理
- 备份和恢复

### 界面说明

#### 1. OpenClaw 配置

**截图说明**：
- JSON 编辑器，带语法高亮
- 左侧显示配置结构树
- 验证按钮和保存按钮

**操作说明**：
- 编辑后点击「验证」检查 JSON 格式
- 验证通过后点击「保存」保存配置
- 自动备份历史版本，可以回滚

#### 2. 访问控制

**截图说明**：
- 用户列表
- 角色列表（管理员/只读/操作员）
- 操作审计日志

**操作说明**：
- **添加用户**: 填写用户名和密码，分配角色
- **修改权限**: 修改用户角色
- **查看审计**: 查看所有用户的操作日志

#### 3. 备份和恢复

**截图说明**：
- 自动备份列表，显示备份时间和大小
- 「立即备份」按钮
- 「恢复」按钮
- 「下载备份」按钮

**操作说明**：
- **创建备份**: 点击「立即备份」
- **恢复备份**: 选择备份 → 点击「恢复」→ 确认
- **下载备份**: 点击下载保存到本地

### 主题设置

支持切换深色/浅色主题：
- 点击顶部导航栏最右侧用户头像 → 主题设置 → 选择深色/浅色/跟随系统

## 常用操作流程图

### 1. 日常巡检流程

```
登录控制台 → 
查看 Gateway Hub 仪表盘 → 
检查 Gateway 状态 → 
检查 Agent 状态 → 
查看模型用量 → 
发现异常 → 查看日志 → 处理问题
```

### 2. 添加新 Agent 流程

```
进入 Settings → 
编辑 openclaw.json → 
添加 Agent 配置 → 
保存 → 
进入 Agent Fleet → 
查看新 Agent → 
测试启动 → 完成
```

### 3. 清理会话流程

```
进入 Session Manager → 
筛选状态:运行中 → 
筛选时间:超过 2 小时 → 
全选 → 
批量终止 → 
筛选状态:完成且超过 30 天 → 
全选 → 批量删除
```

### 4. 月度用量检查流程

```
进入 Model Usage → 
查看本月总用量 → 
检查是否超过配额 → 
查看各模型占比 → 
导出统计数据 → 
成本核算 → 完成
```
